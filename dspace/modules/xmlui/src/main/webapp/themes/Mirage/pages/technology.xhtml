Technology 

Dryad is built upon the open-source DSpace repository software. All customizations not available within the main DSpace distribution are available from the Dryad code repository under an open source (new BSD) license. Information on programmatic access to Dryad's data is available on the Dryad wiki. Those interested in contributing to or using the software are welcome to join the developer mailing list.
